# DSP Lab - Signal Processing

Lab reports and Scilab scripts for the Signal Processing course (HK252), HCMUT - VNUHCM.

## Structure

- **Lab_1/** - Analog/discrete-time signals, sampling, quantization
- **Lab_2/** - Elementary signals (step, impulse, ramp), even/odd decomposition, signal operations
- **Lab_3/** - Scilab functions for delay, advance, fold, addition, multiplication, convolution
- **Lab_4/** - Z-transform, audio/image processing, linear and circular convolution
- **Lab 5/** - Inverse Z-transform, LTI system analysis, Z-transform property proofs

## Tools

- Scilab 2026.0.1
- LaTeX (pdflatex)
